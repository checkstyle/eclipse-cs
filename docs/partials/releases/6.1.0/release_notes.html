Integrates Checkstyle 6.1 plus a few bugfixes. Checkstyle 6.0 has been skipped due to tight release succession and some connected issues.

Please note that as of this version Java 6 is the minimum runtime environment supported. This requires you to run Eclipse on a Java 6 (or newer) JVM in order to use this plugin version.

This restriction does not affect the ability to analyze Java 5 or lower level source code.

Bugfixes
