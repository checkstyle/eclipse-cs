This release integrates the new Checkstyle 5.0 beta2.

A few plugin bugs have been fixed over beta1, as well as a few new features incorporated.
Most notable is likely the Configuration Editor support for custom Checkstyle messages (a feature new in CS 5).

Bugfixes

  • 2057253 Properties missing in Whitespace Around
  • 2097661 Error when not connected to Internet
  • 2542853 Cannot modify project-specific Checkstyle configuration

Features

  • 2143207 Override config properties at fragment build time
  • 1892774 Enhanced Custom Messages
